## 中国优化线路是什么？CN2 GIA / CMIN2 / CUP 一口气说清楚

先把概念搞明白，后面选套餐才不懵。

**CN2 GIA**，全称 China Telecom Next Carrier Network-Global Internet Access，是中国电信骨干网的最高端产品线。路由走AS4809，全程59.43跳，理论上不拥塞、不绕路，晚高峰表现最稳定。这是搬瓦工各类高速套餐的核心线路。

**CN2 GT** 比CN2 GIA低一级。去程部分会走普通公网，只有回程走CN2，晚高峰会有抖动，但总体比普通线路强，价格也便宜得多。

**CMIN2** 是中国移动的高端精品网，AS58807，专门针对移动用户优化。搬瓦工部分套餐支持三网分流，移动用户走CMIN2，延迟相当低。

**CUP（AS9929）** 是中国联通高端线路。搬瓦工的CN2 GIA-E套餐支持CN2 GIA + CMIN2 + CUP三网联合优化——电信走CN2 GIA，移动走CMIN2，联通走AS9929，三条独立通道，互不抢带宽。
